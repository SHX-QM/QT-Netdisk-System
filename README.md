#基于Qt+TCP的简易网盘项目
一个采用C++/Qt开发的CS架构网盘系统，实现客户端与服务端分离，支持用户注册登录注销、文件管理、文件上传下载、目录导航、好友操作、聊天等功能。

##一、项目简介
本项目是一个仿网盘界面与功能的实战项目，基于TCP协议 自定义应用层协议（PDU数据包），实现稳定的文件传输与用户交互。整体分为：
客户端：提供GUI交互、文件列表展示、上传/下载/进入目录/返回上一级/重命名/删除/创建文件夹操作
服务端：处理用户请求、目录管理、文件读写
技术栈：C++、Qt6.10.2、TCP Socket、自定义PDU协议、多文件IO、UTF-8中文支持、sqlite3、单例模式

##二、已实现核心功能
1. 用户系统
用户登录
根目录自动分配（以用户名作为个人空间）
2. 文件管理（核心）
刷新当前目录文件列表
双击进入文件夹
返回上一级目录（根目录禁止返回）
区分文件/文件夹类型显示
支持中文文件名、中文路径、中文目录
3. 文件传输
文件上传（支持图片、压缩包、文档等所有二进制格式）
文件下载
分块读写（4KB 缓冲区），支持大文件稳定传输
基于文件长度校验，保证文件完整不损坏
4. 网络架构
自定义PDU数据包结构
消息类型区分（登录、刷新、进入目录、上传、下载、删除等）
TCP 流式传输
UTF-8编码，跨平台不乱码
5. 交互体验
操作成功/失败提示
目录自动排序（文件夹优先、按名称排序）
路径自动维护与更新
